This is modFriend

Matching Webapp

modFriend was my first ever coding project. modFriend is a matching webapp aimed to match students who were taking the same modules in the semester. It was made as part of an NUS summer programme, Orbital. I worked on modFriend with one other student, using React.js, CSS and Firebase.

We came up with this idea with the COVID-19 situation in mind, where we felt students in NUS were having a harder time to make friends because of lessons being pushed online.

modFriend allows not only users to match with other users, but also has an in-built chat function, allowing users to chat directly on the webapp itself.

Being my first coding project, I was not too familiar with how to start off. Moreover, it was my first time using git, React.js and CSS. This project were my first steps into software engineering, helping me learn how to use basic programs like git.

This project put my problem solving skills to the test, from figuring out how to match students to creating an in-house chatting platform, I put my coding knowledge to work to solve these problems.

modFriend can be accessed at modfriend.vercel.app
