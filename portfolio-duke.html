This is Duke

Chatbot

Duke is a task managing chatbot. It is a green-field individual project done for a Software Engineering module. It was made using Java and JavaFx.

Duke played an integral role in learning the basics of software engineering. It strengthened my coding skills in Java and helped me learn about writing documentation like README and JavaDocs.

Through this project, I also learnt more about writing unit tests with JUnit and the importance of testing.

Duke can be accessed at https://github.com/itsyme/ip/releases/tag/A-Release
